# Git-Workflow-Utilities
The github-utilities Jupyter Notebook consist of a collection of PowerShell and Shell automation scripts to streamline local Git repository management, clean workspaces, and sync multiple projects in parallel.

* **1. Fetching Multiple Repositories at Once**
* **2. Bring an Existing GitHub Repo Local (Clone)**
* **3. Connecting a New Local Project to GitHub (Push Existing)**
* **4. Branching & Workspace Isolation**
* **5. Undoing Changes & Saving Your Work**
* **6. Comparing Changes (Diffing)**
* **7. Viewing History & Collaboration**
* **8. Cleaning Up Untracked Files**
