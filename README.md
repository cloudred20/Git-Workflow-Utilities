# Git-Workflow-Utilities
The github-utilities Jupyter Notebook consist of a collection of PowerShell and Shell automation scripts to streamline local Git repository management, clean workspaces, and sync multiple projects in parallel.

* **Fetching Multiple Repositories at Once**
* **Bring an Existing GitHub Repo Local (Clone)**
* **Connecting a New Local Project to GitHub (Push Existing)**
* **Branching & Workspace Isolation**
* **Undoing Changes & Saving Your Work**
* **Comparing Changes (Diffing)**
* **Viewing History & Collaboration**
* **Cleaning Up Untracked Files**
